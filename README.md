# interactive-learning

把任意文本、链接、文档或问题，转成一条可验证的学习路径，并在必要时动态生成一张 HTML checkpoint card。

## 这是什么

这是一个 `interactive epistemic learning skill`。

它关心三件事：

1. 学什么
2. 怎么学
3. 怎么证明自己学会了

## 这不是什么

- 不是总结器
- 不是默认陪聊器
- 不是默认名人聊天室
- 不是知识卡片生成器
- 不是小型学习网站

## 仓库结构

- `SKILL.md`
  运行时主规则：触发、流程、提问、导出
- `references/design-spec.md`
  checkpoint card 的唯一细节规范来源

## 阅读顺序

1. `README.md`
2. `SKILL.md`
3. `references/design-spec.md`
